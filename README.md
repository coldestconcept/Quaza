# PS5 PKG Tool

Browser-based game dump to PKG converter for jailbroken PS5.

## Features
- Web interface (runs on PS5 via browser)
- Auto-backport from prosperopatches.com
- ShadowMount Plus integration
- LibProsperoPKG-based PKG creation

## Structure
- `payload/` - PS5 ELF payload (HTTP server)
- `www/` - Web interface files
- `python/` - Scraper for prosperopatches
- `docs/` - Documentation
